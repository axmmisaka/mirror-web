Solus 是一个界面美观，对新手友好的 Linux 发行版，使用 `eopkg` 作为包管理器。

*  项目地址：https://getsol.us

### 使用方法

执行以下命令



{% raw %}
<script id="template-0" type="x-tmpl-markup">
{{sudo}}eopkg add-repo Mirror {{http_protocol}}{{mirror}}/packages/shannon/eopkg-index.xml.xz
</script>
{% endraw %}

{% raw %}
